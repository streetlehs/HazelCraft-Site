Frequently Asked Questions

Who can join HazelCraft?

HazelCraft is exclusively for Hazelwick students. Please do not invite outsiders without permission.

What platforms are supported?

Java Edition (cracked & paid): Windows, macOS, Linux
Bedrock Edition: Windows, Linux, iOS, Android, Xbox, PlayStation, Nintendo Switch, Chromebook

Is the server safe?

Yes. Only students and approved staff can join. The server is actively moderated and follows school guidelines to ensure a safe environment.

What gamemode is available?

The server operates in Survival Mode, providing a classic Minecraft experience for all players.

Is HazelCraft endorsed by Hazelwick?

No. HazelCraft is an independent community server created by and for students. It is not officially affiliated with Hazelwick School or its staff.

Are Java and Bedrock players able to play together?

Yes, HazelCraft supports cross-play between Java and Bedrock players. Please note that Bedrock players may experience slightly higher latency.

Is the server modded?

The server features world generation mods (Tectonic, Terralith) and quality-of-life mods (Fast Leaf Decay, VeinMiner, Tree Harvester), but remains largely vanilla.

Will the world ever reset?

The world may be reset with major Minecraft updates, marking a new season for the server.

How do I report a player or problem?

Please contact Leon directly on the Discord Server to report any issues or concerns.

What is the minimum Minecraft version required?

Currently, only version 1.21.5 is supported to ensure the best compatibility for all players.
